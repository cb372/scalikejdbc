ActiveSession

case class ActiveSession(conn: Connection, tx: Option[Tx], isReadOnly: Boolean) extends DBSession with Product with Serializable

Active session implementation of DBSession.

This class provides readOnly/autoCommit/localTx/withinTx blocks and session objects.

import scalikejdbc._

val userIdList = DB autoCommit { session: DBSession =>
  session.list("select * from user") { rs => rs.int("id") }
}
