ScalikeJDBC - Wrapped JDBC for Scala

Just write SQL

This is a thin JDBC wrapper library which just uses java.sql.PreparedStatement internally. Users only need to write SQL and map from java.sql.ResultSet objects to Scala objects. It's pretty simple, really. If you want simple mappers, please also try scalikejdbc-mapper-generator.

https://github.com/seratch/scalikejdbc-mapper-generator
